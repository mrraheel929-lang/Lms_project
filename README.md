# 🎓 Student Learning Management System (LMS)
**Software Construction and Development Lab Final Project**

## 📌 Project Overview
A Python-based LMS designed to manage students, teachers, and course enrollments. Built using OOP principles and verified with PyTest.

## 🛠️ Tech Stack & Tools
- **Language:** Python 3.10+
- **IDE:** VS Code
- **Version Control:** Git & GitHub
- **Testing Framework:** PyTest

## ⚙️ Mandatory Features
- **OOP Design:** Uses Abstract Base Classes, Inheritance, and Encapsulation.
- **Testing:** 20+ Unit tests covering functional edge cases.
- **Modular Structure:** Separated logic (`core/`) and testing (`tests/`).
- **Error Handling:** Robust validation for IDs and data types.

## 🚀 How to Run
1. Clone the repository: `git clone <repo-url>`
2. Install PyTest: `pip install pytest`
3. Run the application: `python main.py`
4. Execute tests: `pytest tests/`
